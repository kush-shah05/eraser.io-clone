This is a Next.js fully functional clone of Eraser.io using Next.js, React, TypeScript, and Tailwind CSS. built and deployed a collaborative drawing application, covering essential concepts in a full-stack app.

Open  with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
